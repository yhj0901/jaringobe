**주의 항목 (버그 아님)**
- `COOKIE_SECURE=false` 가 로컬 기본값 — **배포 시 true 필수** (.env.example 주석 명시됨)
- rate limit 인메모리 — 멀티 인스턴스 배포 시 Redis 교체 (architecture.md 확장점 기록됨)

**결론: 보안 테스트 10/10 PASS, 취약점 발견 0건**
